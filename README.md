# 🎬 IPSO Detector

AI which can detect IPSO with fake news about ukraine war.

## ✨ Technologies

- `Django`

## 🚀 Features

- AI which can detect fake news
 
## 📍 The Process

I decided to create a tool which can help me and other people to detect fake news which is very often happens in nowadays by Russians, which can publish fake posts, mostly in telegram.

## 🚦 Running the Project

1. Clone the repository
2. Create virtual environment `python3 -m venv venv`
3. Activate virtual environment `surce venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
